# PVR-001: Use a mobile-shaped popup shell

Status: accepted

Direction A is the selected visual direction for the popup vocabulary review. The design uses a mobile-first 390x844 composition with a calm editorial hierarchy, black-white-orange tokens, large touch targets, vertical Settings scrolling, and explicit Learn, flashcard front, flashcard back, and Settings states. The outer rounded phone frame exists only in the mockup; production UI should use the same responsive composition inside the browser popup and remain portable to a future mobile app shell.

Related design mockup: `docs/features/PVR-001-design-A-mobile.html`.
