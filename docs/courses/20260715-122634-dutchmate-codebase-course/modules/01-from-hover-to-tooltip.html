From Hover To Tooltip

Start with the user experience: you hover a Dutch word, and a tiny translation appears without breaking your reading flow.

What this app actually is

DutchMate is a browser extension for people reading Dutch websites while thinking in English and Telugu.

The key promise is simple: keep reading the web, and let translation come to you instead of forcing you into a new tab.

🖱️

Hover mode

Pause over a single word and DutchMate quietly checks whether a tooltip should appear.

✂️

Selection mode

Select a short phrase or sentence when one word is not enough.

📚

Learning first

This is not a generic translator. It is tuned for a Dutch, English, Telugu learning triangle.

💡
Why this matters for steering AI

If you can describe the product as "a context-aware translation tooltip inside a browser extension," AI tools stop suggesting the wrong architecture, like building a full website or chat app instead.

The core journey is a relay race

Think of DutchMate like a relay team. Each runner touches the baton briefly, does one job, and passes it on fast.

1
You hover or select text

The web page is the track. DutchMate notices a real reading action, not a random background event.

2
The content script packages the request

It captures the text, location, and context like hover or selection.

3
The background worker asks for translation

That keeps network work out of the page-facing layer.

4
The tooltip renders the answer

The reader sees a tiny result right where their eyes already are.

Where the page wires itself up

This snippet from [src/content/index.ts](/home/mgurram/MGurramAI/projects/dutchmate-proj/dutchmate/src/content/index.ts:54) is the stage manager. It creates the lookup module, the tooltip view, and all the page event listeners.

CODE 
let currentSettings = defaultSettings;
const lookupModule = new WebpageLookupModule({
  getSettings: () => currentSettings,
  transport: createRuntimeLookupAdapter({
    browserTarget: __BROWSER_TARGET__,
    chromeDirectTranslationFallbackMs: CHROME_DIRECT_TRANSLATION_FALLBACK_MS,
    directTranslationTimeoutMs: DIRECT_TRANSLATION_TIMEOUT_MS,
    extensionApi,
    getSettings: () => currentSettings,
    delay,
  }),
  runWithTimeout: withTooltipTranslationTimeout,
  tooltipTimeoutMs: TOOLTIP_TRANSLATION_TIMEOUT_MS,
});
const tooltipView = createTooltipViewAdapter(() => {
  void lookupModule.handleSaveAction();
});
const lifecycleController = createWebpageLifecycleController({
  getSettings: () => currentSettings,
  lookupModule,
  tooltipView,
});
document.addEventListener("mousemove", lifecycleController.handleMouseMove, { passive: true });
