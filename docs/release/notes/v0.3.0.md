# DutchMate 0.3.0

This release adds the first local flashcard review loop for saved Dutch vocabulary. Review data remains private and browser-local, with no account, cloud sync, or server-side vocabulary storage.

## Added

- Added a popup Learn view with due-word, new-word, and all-word review flows.
- Added Dutch, English, and Telugu flashcard views with Show Answer and Again, Hard, Good, and Easy ratings.
- Added simple local review scheduling: 24 hours for Again/Hard, 3 days for Good, and 7 days for Easy.
- Added review progress, completion states, optional page context, card direction, and a due-review toolbar badge.
- Added versioned local vocabulary backup import/export and safe clear behavior.
- Added cache controls for hovered and selected words while keeping translation cache separate from saved vocabulary.

## Fixed

- Fixed cross-browser popup sizing, generated extension entry points, and packaged manifest verification.
- Fixed saved vocabulary synchronization across reading pages, the popup, and the Options page.
- Fixed preservation and display of Telugu meanings in canonical review cards.

## Changed

- Added compact Dutch-English-Telugu saved-vocabulary tables with source-language indication and delete controls.
- Clarified that the Options page owns general extension settings while popup Settings focuses on review preferences and routing.
- Kept cached translation entries out of learner-controlled vocabulary backups because they are lookup acceleration data, not study cards.

## Verification

- `corepack pnpm verify`
- `corepack pnpm verify:release`
- Chrome and Firefox browser smoke testing

## Notes

- No new browser permissions.
- No account, cloud sync, or backend provider change.
- Saved vocabulary, review metadata, page context, and translation cache remain local browser data.
